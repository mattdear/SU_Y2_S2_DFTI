Using the Edward2 server outside university

The Neptune (EPHP) server is only available inside university. However you can use an alternative server - Edward2 - from home. To use this you need to download FileZilla to your home machine. It's available here:

  • Windows 64 bit
  • Windows 32 bit
  • Go here for Mac or Linux. Avoid the "recommended" links. These give you the option of installing other software which has in some cases been reported as suspect.

When using FileZilla, enter the following details for Edward2:

  • Host: edward2ftp.solent.ac.uk
  • Username and Password: your EPHP login details
  • Port: 990
Upload to the public_html folder.

Install a PHP system at home

As an alternative, if you are using Windows you can install XAMPP - a complete PHP/Apache/MySQL/PHPMyAdmin system which you can use to develop PHP applications at home. For instructions see here. There are security risks involved in setting up a web server on a home PC, so please read the instructions carefully!

Linux setup of an Apache server with PHP and MySQL is generally easy if you're used to the command line - ask me about this. For Mac, MAMP is available which is similar to XAMPP.
